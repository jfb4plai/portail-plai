⏇ 3 minutes à lire 🎓 Tous niveaux, tous réseaux

Un skill, c'est un texte que vous copiez-collez une fois dans Claude. Après ça, Claude sait comment travailler avec vous : il connaît votre niveau d'enseignement, vos référentiels FWB, les Aménagements Universels. Il produit des documents utilisables le lendemain matin — pas des réponses génériques pour n'importe quel enseignant dans n'importe quel pays.

Exemple concret
Vous écrivez :
« /plan — fractions pour P4, 50 minutes, iPads disponibles »

Claude produit : un plan en 5 phases avec timing précis (5+12+10+15+8 min), les consignes à lire aux élèves phase par phase, la liste du matériel, une grille d'observation avec 3 comportements à observer.

⚠️ RGPD
2. Ce qu'il faut savoir avant de commencer
Ces skills préparent vos séances. Ils ne s'adressent pas à vos élèves — cette distinction compte du point de vue du RGPD (Règlement Général sur la Protection des Données, applicable en Belgique).
Règle 1 — Aucune donnée d'élève identifiable dans Claude

Noms, prénoms, dates de naissance, classe, établissement : ne jamais les saisir. Si vous décrivez un élève, utilisez « Élève A » ou un prénom fictif. Les bulletins, rapports PMS et résultats d'évaluations nominatifs ne vont pas dans Claude.

Règle 2 — Désactiver l'entraînement du modèle dès l'inscription

Sur le plan gratuit, Anthropic peut utiliser vos conversations pour améliorer ses modèles — sauf si vous désactivez cette option. À faire une seule fois, juste après l'inscription : Paramètres (⚙️) → Confidentialité → désactiver « Améliorer Claude pour tous » → Enregistrer.

Règle 3 — Vous validez avant de distribuer

Claude produit des bases de travail. Vous restez responsable de ce que vous donnez à vos élèves. Relisez, vérifiez l'alignement sur les référentiels FWB (enseignement.be), corrigez si besoin.

🎓 Installation
3. Comment installer un skill — 4 étapes
Première installation : 10 à 15 minutes. Ensuite : 2 minutes par skill.
1
Créer un compte Claude.ai gratuit
Aller sur claude.ai → « Commencer gratuitement » → s'inscrire avec une adresse e-mail ou un compte Google. Pas de carte bancaire.
2
Créer un Projet
Les Projets permettent d'enregistrer des instructions une fois pour toutes — gratuits depuis février 2026. Barre de gauche → « Projets » → « + Nouveau projet » → nommer (ex. « Skills pédagogiques PLAI ») → « Modifier les instructions du projet » → coller le contenu du skill choisi → « Enregistrer ».
3
Utiliser le skill
Dans le Projet, ouvrir une conversation. Taper le déclencheur du skill, coller votre contenu. Claude s'occupe du reste.
4
Relire avant d'imprimer
Claude se trompe. Pas souvent sur la structure, parfois sur les détails disciplinaires. Toujours vérifier.
Conseil

Un Projet par skill, ou tous les skills dans un seul Projet, l'un après l'autre — séparez-les par une ligne vide et un titre clair comme « SKILL : plan-seance-fwb ».

4. Les 3 skills disponibles
Pour chaque skill : le déclencheur, la recherche mobilisée, un exemple concret, et le lien vers le fichier à copier.
/plan-seance-fwb📋 Plan de séance FWB
Déclencheur : « /plan » ou « prépare-moi une séance sur [sujet] pour [niveau] ». Construit un plan complet en 5 phases temporisées : mise en situation, exploration, structuration, entraînement, clôture. Objectif formulé avec un verbe observable. Consignes rédigées pour être lues directement aux élèves. Grille d'observation avec comportements concrets. Fonctionne pour toute la FWB : maternelle (M1–M3), primaire (P1–P6), secondaire général, technique, professionnel (S1–S7), tous réseaux.
Exemple — « /plan — multiplication par 10 et 100, P3, 50 minutes, 22 élèves, iPads disponibles » → objectif Socles P3 Maths, 5 phases chronométrées, consignes à lire aux élèves, liste du matériel, grille d'observation à 3 comportements.
Recherche : Bloom & Krathwohl (2001) — taxonomie révisée. Rosenshine (2012) — enseignement explicite (réel, hors corpus RISS). Vygotsky (1978) — Zone Proximale de Développement.

/seance-differenciee🔵🟢🟠 Séance Différenciée
Déclencheur : « différencie », « /diff », « 3 niveaux », « adapte pour tous mes élèves ». Transforme n'importe quel contenu pédagogique en 3 versions : Niveau A (accès sécurisé, Aménagements Universels intégrés), Niveau B (standard du programme), Niveau C (approfondissement qualitatif — pas juste « plus d'exercices »). L'objectif de compétence reste le même dans les 3 versions. Le guide enseignant précise l'organisation de classe, le timing et les signaux d'alerte.
Exemple — « différencie ce texte — S2 Français » + [texte collé] → version A phrases courtes + AU, version B restructurée format FWB, version C interdisciplinaire niveau Bloom 4-6, guide enseignant.
Recherche : Tomlinson (1999/2014) — différenciation pédagogique (réel, hors corpus RISS). UDL / CUA (Rose & Meyer 2002) — base des Aménagements Universels FWB. Vygotsky (1978) — ZPD et étayage.

/rubrique-qualite-output✅ Rubrique Qualité Output
Déclencheur : « /qualite », « /check-output », « évalue cet output », « c'est utilisable en classe ? ». Méta-skill : vérifie si un output produit par un autre skill est utilisable. Score sur 5 critères communs (utilisabilité, alignement FWB, qualité pédagogique, contexte PLAI, absence d'erreurs) et des critères propres à chaque skill. Verdict 🟢 Utilisable / 🟠 À retravailler / 🔴 Non utilisable, avec les corrections à faire et, si nécessaire, un prompt amélioré à copier-coller.
Exemple — « /qualite » + [output du skill /plan-seance-fwb collé] → score par critère, verdict avec seuils expliqués, corrections concrètes, nouveau prompt si 🔴.
Recherche : Black & Wiliam (1998) — feedback formatif. Hattie & Timperley (2007) — conditions d'un feedback efficace. Rosenshine (2012) — vérification de la compréhension (réel, hors corpus RISS).

Questions fréquentes
Le plan gratuit est-il vraiment suffisant ?

Oui. Les Projets (avec instructions personnalisées) sont gratuits depuis février 2026. La limite du plan gratuit, c'est le nombre de messages par session — suffisant pour préparer une séance. Si vous atteignez la limite, attendez 5 heures ou continuez dans une nouvelle conversation.

Puis-je mettre plusieurs skills dans le même Projet ?

Oui. Collez les contenus SKILL.md les uns après les autres dans les instructions du Projet. Claude activera le bon skill selon le déclencheur que vous tapez. Séparez-les par une ligne vide et un titre clair comme « SKILL : plan-seance-fwb ».

Ce que Claude produit est-il toujours juste ?

Non. Sur la structure et les phases, il est solide. Sur les détails disciplinaires et les références aux référentiels FWB, il peut se tromper. Relisez toujours avant d'utiliser en classe. Pour les référentiels, vérifiez sur enseignement.be avant tout usage en évaluation certificative.

Puis-je partager les documents produits avec des collègues ?

Les documents, oui. Votre compte Claude, non — chaque enseignant crée le sien. Un compte gratuit prend deux minutes.

Et si je veux utiliser Claude directement avec mes élèves ?

C'est un autre sujet, avec d'autres règles RGPD (consentement parental, données de mineurs, finalité pédagogique déclarée). Ces skills sont des outils de préparation pour l'enseignant. Une ressource sur l'usage de l'IA avec les élèves est en cours de développement au Pôle.
